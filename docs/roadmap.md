# Curv Roadmap

## v0.1.0 — Foundation

- Browser-based JSON formatter
- JSON minifier
- Validation workflow
- Better validation messages where available
- File upload
- Output download
- Copy and clear actions
- Text statistics
- Basic JSON structure analysis
- Responsive dark interface

## v0.2.0 — Editor Experience

- Syntax highlighting
- Better editor keyboard behavior
- Auto-format on paste
- More detailed validation panel
- Theme preferences

## v0.3.0 — Tree View

- Expandable tree viewer
- Copy JSON path
- Copy node value
- Search keys and values

## v0.4.0 — Diff

- Side-by-side JSON diff
- Semantic JSON comparison
- Ignore key order option

## v0.5.0 — Converters

- JSON to YAML
- JSON to CSV
- JSON to XML
- JSON to Markdown table
- JSON to Java records/classes
- JSON to SQL helpers
