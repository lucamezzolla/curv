# Privacy

Curv is designed as a privacy-first JSON toolbox.

## Local Processing

Curv processes JSON locally in the browser.

The application does not require a backend for its core features.

## Sensitive Data

Even when using local-only tools, be careful with sensitive data such as:

- API keys
- Access tokens
- Customer data
- Production payloads
- Internal configuration files

## Future Desktop Version

The planned desktop version will keep the same privacy-first approach and will focus on local file workflows.
